Controls for creating oulines (like a tree control).

w: namespace -- Namespace aware templates typically refer to these components with the "w:" prefix (e.g.{@code })

Note that while {@link ariba.ui.outline.OutlineControl w:OutlineControl} can be used in concert with {@link ariba.ui.outline.OutlineRepetition w:OutlineRepetition}, it is most often used in the first column of an {@link ariba.ui.table.AWTDataTable t:DataTable} to create an outline table. @see ariba.ui.table#AWTDataTable
