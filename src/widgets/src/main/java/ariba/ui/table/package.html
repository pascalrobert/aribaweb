Family of components for creating table (grid) controls (including scrolling flat tables, outline (grouped) tables, outline (tree) tables, and multi-dimensional pivot tables).

t: namespace -- Namespace aware templates typically refer to these components with the "w:" prefix (e.g. {@code , })

Commonly used tags include:

  • {@link ariba.ui.table.AWTDataTable t:DataTable}
  • {@link ariba.ui.table.AWTColumn t:Column}
  • {@link ariba.ui.table.AWTHeadingArea t:HeadingArea}
  • {@link ariba.ui.table.AWTButtonArea t:ButtonArea}
  • {@link ariba.ui.table.AWTSelectionButtonArea t:SelectionButtonArea}

The DataTable list state is managed by the {@link ariba.ui.table.AWTDisplayGroup}. For Outline Tables, outline expansion / selection state is managed by the {@link ariba.ui.outline.OutlineState} and for Pivot Tables the pivot grouping state is managed by the {@link ariba.ui.table.AWTPivotState}. @see ariba.ui.outline
